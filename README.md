# DBF to CSV Converter

This project allows you to convert legacy DBF files (dBase/FoxPro) into modern CSV format for further processing or database import (e.g. Supabase).

## Features

- Supports multiple DBF files
- Column selection
- Basic data cleaning
- UTF-8 CSV export

## Tech Stack

- Python
- pandas
- dbfread

## Usage

```bash
python convert_dbf_to_csv.py
```
